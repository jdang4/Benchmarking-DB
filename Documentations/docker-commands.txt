Running Benchmark

sudo docker run -it --rm --name jonathan-client --link postgres-master:bitnami/postgresql --link redis-master:bitnami/redis -v ~/docker/Benchmark/stats:/benchmark/stats jonathan-cpp



Running Redis Client

sudo docker run -it --rm --name jonathan-client --link redis-master:bitnami/redis jonathan-cpp



Redis Master

sudo docker run --name redis-master -v /var/lib/redis:/bitnami/redis/data -e REDIS_REPLICATION_MODE=master -e ALLOW_EMPTY_PASSWORD=yes --rm -d bitnami/redis:latest



Redis Replica

sudo docker run --name redis-replica -v /home/jonathand/redis-data:/bitnami/redis/data --link redis-master:master -e REDIS_REPLICATION_MODE=slave -e REDIS_MASTER_HOST=master -e ALLOW_EMPTY_PASSWORD=yes --rm -d bitnami/redis:latest



Without AOF Persistence

-e REDIS_AOF_ENABLED=no
